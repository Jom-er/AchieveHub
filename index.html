Points

1. Late Submission Pass 📖
🔹 Cost: 50 – 100 points
🔹 Rules: Allows a student to submit an assignment 1–3 days late without penalty.
🔹 Sample Point System:

  • 1-day extension → 50 points
  • 2-day extension → 75 points
  • 3-day extension → 100 points

2. Extra Study Materials 📚
🔹 Cost: 30 – 150 points (depends on material type)
🔹 Rules: Unlock access to additional notes, summaries, or practice tests.
🔹 Sample Point System:

  • Lecture Notes → 30 points
  • Exam Study Guide → 75 points
  • Practice Quizzes & Sample Answers → 100 points
  • Premium Study Package (All of the above) → 150 points

3. Retake a Quiz/Test 📝
🔹 Cost: 200 – 500 points
🔹 Rules: Allows a student to retake a failed quiz/test for a better score.
🔹 Sample Point System:

  • Retake a Small Quiz → 200 points
  • Retake a Midterm Test → 350 points
  • Retake a Final Exam → 500 points (Limited use per semester)

Bonus: Earning Points System (Example)

  • Attending a class → 5 points per session
  • Participating in discussions → 10 points per class
  • Scoring high on an exam → 50 points for A grade
  • Helping a fellow student → 10-30 points (peer assistance program)

Note: The points system can be updated periodically based on feedback and new rules.
